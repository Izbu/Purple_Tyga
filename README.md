# Purple Tyga — Official Website

> *Straight from the Urban Jungle. Viva el Cartel.*

The official digital headquarters of **Purple Tyga** — streamer, content creator, and voice of the cartel. A cinematic, dark-themed personal brand website built to represent the culture.

---

## Features

- Full-screen intro video with skip button
- Hero slideshow with 7 custom visuals
- Stream highlights linked to YouTube
- Spotify now playing section
- Social links hub (YouTube, Instagram, Twitch, Spotify, Discord)
- Contact form with direct email integration
- Road Rage custom font branding
- Framer Motion animations throughout
- Fully responsive

---

## Tech Stack

- **React** + **TypeScript**
- **Vite**
- **TailwindCSS**
- **Framer Motion**
- **TanStack Router**
